# Audi TT Mk1 — Big Turbo Build

**Platform:** Audi TT Mk1 | 1.8T  
**Location:** Philadelphia, PA — Home Shop  
**Power Target: ~450 whp**  
**Status: Active Build**

-----

## Build Goal

A full big-turbo build targeting approximately **450 wheel horsepower** on the 1.8T platform. Built in a home shop with a focus on proper engineering — forged internals, upgraded head components, and a purpose-built turbo setup.

## Completed Work

**Starter Replacement**
Diagnosed intermittent no-crank condition traced to a failing starter. Replaced with new unit — fault resolved.

**Power Steering Rack**
OEM replacement rack sourced and on hand. Installation planned.

**Timing Belt Kit Prep**
Gates timing belt kit acquired. Job requires:

- VAG-specific cam locking tools
- Crank TDC positioning tool
- Engine supported from below
- Passenger-side engine mount removed
- Coolant drained

**Exhaust Manifold Modification**
Existing OEM manifold to be modified with a V-band flange for clean turbo fitment on the GTX3076R.

## About

The TT is the long-term flagship build under [Neif Design](https://neifdesign.com). While the MK7 Jetta is the daily driver project, the TT is where the real engineering starts — a purpose-built platform designed to showcase what’s possible with proper planning and fabrication.

The goal isn’t just a fast car. It’s a documented engineering project that demonstrates the full build process from concept to dyno.
